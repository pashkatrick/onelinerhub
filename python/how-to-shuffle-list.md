# How to shuffle list

```python
import random

list = [1, 2, 3]
random.shuffle(list)

```

- `import random` - library to work with randomizing
- `list` - sample list to shuffle
- `random.shuffle` - will shuffle specified list (list will be overwritten)

## Example: 
```python
import random

list = [1, 2, 3]
random.shuffle(list)

print(list)
```
```
[3, 1, 2]

```
