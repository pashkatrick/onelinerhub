# How to order a list

```python
ordered = sorted(list)
```

- `list` - example list
- `ordered` - will contain sorted list
- `sorted(` - function will [sort elements in list](https://docs.python.org/3/howto/sorting.html) and return new list

group: order

## Example: 
```python
list = [10, 2, 7]
ordered = sorted(list)
print(ordered)
```
```
[2, 7, 10]

```
