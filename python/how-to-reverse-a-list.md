# How to reverse a list

```python
list = ['a', 'b', 'c', 'd']
reversed = list[::-1]
```

- `list` - sample list to reverse
- `reversed` - will contain reversed list
- `[::-1]` - reverses list

## Example: 
```python
list = ['a', 'b', 'c']
reversed = list[::-1]
print(reversed)
```
