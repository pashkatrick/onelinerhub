# How to left trim a string

```python
" text ".lstrip()
```

- `" text "` - example string to trim whitespaces
- `.lstrip()` - will trim whitespaces from left side of the string

group: trim

## Example: 
```python
str = " text "
trimmed = str.lstrip()
print(f'Original: "{str}", trimmed: "{trimmed}"')
```
```
Original: " text ", trimmed: "text "

```

## Additional keywords
- strip
- remove whitespaces
