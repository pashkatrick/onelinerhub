# How to split string by colon

```python
str = 'a:b:c:d'
list = str.split(':')
```

- `a:b:c:d` - sample string to split
- `str.split(` - will split string by specified character
- `':'` - split string by colon

group: split_string

## Example: 
```python
str = 'a:b:c:d'
print(str.split(':'))
```
```
['a', 'b', 'c', 'd']

```
