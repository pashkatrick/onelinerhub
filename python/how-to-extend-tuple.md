# How to extend tuple

```python
extended = (1, 2, 3) + (4,)
```

- `extended` - variable will contain extended tuple
- `(1, 2, 3)` - original tuple to extend
- `(4,)` - single-value tuple to add to original tuple

## Example: 
```python
extended = (1, 2, 3) + (4,)
print(extended)
```
