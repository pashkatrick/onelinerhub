# How to check if object is of certain type

```python
isinstance(object, type)
```

- `isinstance(` - returns `True` if `object` if of `type` type
- `object` - object to check type of
- `type` - type to check `object` against

group: object_type

## Example: 
```python
obj = 'hi'
print(isinstance(obj, str))
```
```
True
```
