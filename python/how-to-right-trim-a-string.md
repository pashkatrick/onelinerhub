# How to right trim a string

```python
" text ".rstrip()
```

- `" text "` - example string to trim whitespaces
- `.rstrip()` - will trim whitespaces from right side of the string

group: trim

## Example: 
```python
str = " text "
trimmed = str.rstrip()
print(f'Original: "{str}", trimmed: "{trimmed}"')
```
```
Original: " text ", trimmed: " text"

```

## Additional keywords
- strip
- remove whitespaces
