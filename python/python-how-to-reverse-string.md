# How to reverse string

```python
str = 'test'
reversed = str[::-1]
```

- `str =` - test string to reverse
- `reversed` - reversed string
- `[::-1]` - reverses string

## Example: 
```python
str = 'test'
reversed = str[::-1]
print(reversed)
```
