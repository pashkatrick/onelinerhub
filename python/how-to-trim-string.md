# How to trim a string

```python
" text ".strip()
```

- `" text "` - example string to trim whitespaces
- `.strip()` - will trim whitespaces from both sides of the string

group: trim

## Example: 
```python
str = " text "
trimmed = str.strip()
print(f'Original: "{str}", trimmed: "{trimmed}"')
```
```
Original: " text ", trimmed: "text"

```
